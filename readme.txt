== Description ==

Vanilla is the simple theme for blogging. And for single column modern websites.

* Mobile-first, Responsive Layout
* Custom Drawer Menu
* Custom Colors
* Custom Background
* Custom Logo


== Installation ==

= Installation using "Add New Theme" =
1. From your Admin UI (Dashboard), go to Appearance => Themes. Click the "Add New" button.
2. Search for theme, or click the "Upload" button, and then select the theme you want to install.
3. Click the "Install Now" button.

= Manual installation =
1. Upload the "vanilla" folder to the "/wp-content/themes/" directory.

= Activiation and Use =
1. Activate the Theme through the "Themes" menu in WordPress.
